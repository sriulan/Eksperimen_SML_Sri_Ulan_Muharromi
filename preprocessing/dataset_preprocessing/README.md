# Dataset Preprocessing

Dataset preprocessing dibuat dari Rice Image Dataset menggunakan script:

* automate_Sri_Ulan_Muharromi.py

Dataset hasil preprocessing tidak disertakan dalam repository karena terdiri dari sekitar 75.000 file gambar. Dataset digunakan secara lokal untuk proses pelatihan model machine learning.

Dataset sumber:
https://www.kaggle.com/datasets/muratkokludataset/rice-image-dataset
