# Dataset Raw

Dataset yang digunakan pada proyek ini berasal dari Rice Image Dataset.

Sumber dataset:
https://www.kaggle.com/datasets/muratkokludataset/rice-image-dataset

Dataset tidak disertakan dalam repository karena terdiri dari sekitar 75.000 file gambar. Untuk menjaga repository tetap ringan dan mudah diakses, dataset dapat diunduh langsung dari sumber Kaggle di atas.

Dataset digunakan pada tahap eksplorasi data, preprocessing, dan pelatihan model.
